Actividad 7 - Uso de arreglos
Este programa sirve para guardar las notas de varios estudiantes usando arreglos. Se ingresan 4 notas por cada estudiante y el programa verifica que estén entre 0 y 100. Luego calcula los promedios y muestra la nota más alta la más baja y cuántos estudiantes pasaron o no.
Como se puede ver en las imágenes al ingresar la cantidad de estudiantes el programa pide las 4 calificaciones de cada uno después calcula el promedio y al final muestra el promedio general la nota más alta la más baja y la cantidad de aprobados y reprobados.
